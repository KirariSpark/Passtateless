# Passtateless

能够自定义密码生成方式的无状态密码管理器

![home.png](readme_assets/home.png)

## ⚠️ 严重安全警告

本项目仅为个人课程作业，不建议用于真实的密码存储场景！

- 本人非密码学或信息安全专业，未经过专业的安全审计，无法保证应用在加密逻辑、内存清理、防逆向等方面不存在致命漏洞。
- 如果你确实需要管理密码，请使用业界成熟的密码管理器。
- 本人不保证对此项目的持续维护，维护随时可能终止

风险自负！
