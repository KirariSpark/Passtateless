### 单个密码条目字典结构

```json
{
  "identifier": "用于标识这一密码条目的名称，显示给用户看",
  "userName": "用户名，也可以是其他用于标识唯一的账号的信息",
  "account": "当前密码对应的账号",
  "starred": false
}
```

`starred`标记该配置是否被收藏。  
生成预设和自定义配置不会被存储。

### 多个密码条目字典结构

```json
{
  "folder1": [{...}, {...}], // 被归类的记录
  "folder2": [{...}, {...}], // 另一组归类的记录
  "": [{...}] // 未被归类的零散记录
}
```

### 生成配置条目结构

```json
[
  {
    "name": "第一个操作的名称",
    "args": ["第一个操作的参数"]
  },
  {
    "name": "第二个操作的名称",
    "args": ["第二个操作的参数"]
  },
  {
    "name": "第三个操作的名称"
  }
]
```

若没有参数，可以不传`args`  
`name`的值均和`modules\generator\core.dart`中的方法相同
