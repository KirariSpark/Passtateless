# 常见问题解答

## 编辑档案时，“移除XXX”功能有什么用？

通常用于临时更改密码，使其符合某些规则。它会在生成后进行处理，这代表如果你开启了它，密码长度可能会缩短。  
通常来说，如果需要持续使用，你应该在生成配置中添加相应功能。

## 为什么不保存生成预设？

生成预设决定了你的密码将被怎么计算，如果你的主密码泄漏，不被保存的生成预设将是最后一道防线。如果保存生成预设，那么你的密码将完全泄漏。  
如果你忘记了预设，我们也无法为你恢复。

## 我怎么记录我的密码？

由于无状态特性，Passtateless 无法记忆你的密码。它能做的只有在你需要时计算出一个可复现的密码。  
因此，你需要在在 Passtateless 中添加了记录后，同步更改对应账号的密码为 Passtateless 为你生成的版本。

## 生成的密码会变化吗？

不会。生成结果是完全可复现的，只要你的输入不变，生成的密码就不会发生变化。

## 忘记了主密码怎么办？

如果你忘记了主密码，你就永远失去了你的数据，我们无法为你找回或恢复数据 —— 你的数据已被使用 AES256 加密。  
你只能清除 Passtateless 的数据以继续使用它。  
请始终牢记你的主密码。
