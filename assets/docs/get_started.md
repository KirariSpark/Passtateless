# 开始使用

Passtateless 的使用流程极其简洁，只需几步即可完成密码档案的创建与获取。

### 1. 进入密码管理

在应用主页，点击 **“全部密码”** 模块，进入你的密码资料夹列表。  
选择或新建一个资料夹，进入档案列表。

### 2. 新建档案

点击列表页面右下角的按钮，选择“新建记录”，即可新建一个档案，并自动进入档案的编辑页面。

### 3. 填写基本信息

在编辑页面中，你可以根据需要键入该账户的基础信息，例如：

* **档案名**：用于标识这个档案的用途  
* **用户名**：用于标识这个账号的用户名，或其他能标识一个用户的内容  
* **账号/邮箱**：用于标识这个账号所在的平台、邮箱，或其他能标识一个账号或平台的内容  

**核心安全机制**  
在 Passtateless 中，你的数据受到双重保护：
1. **静态数据加密**：你所保存的档案名、用户名等基础信息，会在本地使用你的主密码结合业界标准的 **AES-256** 加密算法进行高强度加密存储。没有你的主密码，这些档案数据在物理层面只是一堆无法读取的乱码。
2. **生成逻辑不落盘**：用于计算密码的**主密码**以及你配置的**自定义生成算法绝对不会被记录或保存**。这意味着，即使攻击者通过某种极端手段获取了你的主密码并解密了档案，他也难以算出你的真实登录密码——因为他不知道你使用了哪种 JSON 算法。

### 4. 保存档案

编辑完成后，返回档案列表或者资料夹列表，点击右下角的按钮，选择“保存更改”，即可保存你的更改。  
由于加密保存并不快，选择自动保存可能会影响性能和体验，因此本软件采用手动保存。

### 5. 配置完成后

**收藏置顶**  
在“密码管理”列表页面中，你可以选择将常用的档案项设置为 **“收藏”**。  
收藏的项目会直接展示在应用主页的顶部区域，让你无需进入列表即可快速访问。

**获取密码**  
在列表中点击任意一个档案项，将进入只读的查看界面。在这个界面中，你可以查看档案配置而无法修改它。  
你只需点击 “复制” 按钮，Passtateless 就会根据你的输入信息和预设算法在后台计算出密码，并自动将生成的密码复制到剪贴板。  

**查看密码**  
不需要使用，但需要记忆密码？使用 “查看密码” 按钮。  
它会跳转到专门显示密码的页面，将你的密码完整地显示出来。  
该页面使用等宽字体 Source Code Pro，你无需担心分不清相似字符。
