# 开始使用

Passtateless 的使用流程极其简洁，只需几步即可完成密码档案的创建与获取。

### 1. 进入密码管理

在应用主页，点击 **“全部密码”** 模块，进入你的密码资料夹列表。  
选择或新建一个资料夹，进入档案列表。

### 2. 新建档案

点击列表页面底部的 **“+”** 按钮，即可进入档案的编辑页面。

### 3. 填写基本信息

在编辑页面中，你可以根据需要键入该账户的基础信息，例如：

* **档案名**：用于标识这个档案的用途  
* **用户名**：用于标识这个账号的用户名，或其他能标识一个用户的内容  
* **账号/邮箱**：用于标识这个账号所在的平台、邮箱，或其他能标识一个账号或平台的内容  

**核心安全机制**  
在 Passtateless 中，你的数据受到双重保护：
1. **静态数据加密**：你所保存的档案名、用户名等基础信息，会在本地使用你的主密码结合业界标准的 **AES-256** 加密算法进行高强度加密存储。没有你的主密码，这些档案数据在物理层面只是一堆无法读取的乱码。
2. **生成逻辑不落盘**：用于计算密码的**主密码**以及你配置的**自定义生成算法绝对不会被记录或保存**。这意味着，即使攻击者通过某种极端手段获取了你的主密码并解密了档案，他也难以算出你的真实登录密码——因为他不知道你使用了哪种 JSON 算法。

### 4. 保存档案

编辑完成后，无需寻找保存按钮，只需点击界面的 **“返回”** 键，系统便会自动为你加密并保存当前编辑的内容。

### 5. 配置完成后

**收藏置顶**  
在“密码管理”列表页面中，你可以选择将常用的档案项设置为 **“收藏”**。  
被收藏的项目会直接展示在应用主页的顶部区域，让你无需进入列表即可快速访问。

**获取密码**  
在列表中点击任意一个档案项，将进入**只读的查看界面**。在这个界面中，你可以查看档案配置而无法修改它。  
你只需点击 **“复制”** 按钮，Passtateless 就会根据你的输入信息和预设算法在后台计算出密码，并自动将生成的密码复制到剪贴板，随后直接去目标网站粘贴登录即可。
