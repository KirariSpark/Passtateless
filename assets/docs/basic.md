# 基本信息

欢迎了解 Passtateless —— 一款无状态密码管理器。本节将帮助你了解 Passtateless 的核心理念与特色功能。

## 什么是无状态密码管理器

无状态密码管理器是一种不保存、不存储任何密码数据的密码管理工具。  
与传统密码管理器（如 1Password、Bitwarden）需要将密码库加密后保存在本地或云端不同，Passtateless 的核心逻辑是计算而非存储。  
当你需要登录某个网站时，Passtateless 会通过特定的算法实时计算出该网站的密码。  

这意味着：
* **零数据丢失风险**：因为没有数据库，所以不存在密码库被黑客攻击、窃取或因设备损坏而丢失的风险。
* **无需同步**：只要你在任何设备上安装了 Passtateless 并输入相同的主密码和网站信息，就能随时计算出完全相同的密码。
* **真正的私密**：你的密码永远只存在于你的脑海中，不经过任何第三方服务器。

## 什么是主密码

主密码是你在 Passtateless 中唯一需要记忆的密码。它是生成你所有其他账户密码的“万能钥匙”。  

在 Passtateless 中，主密码具有以下特点：
* **不可恢复**：由于我们不存储任何数据，如果你忘记了主密码，将无法找回你生成的任何网站密码。请务必将其牢牢记在脑海中。
* **至关重要**：主密码的强度直接决定了你所有衍生密码的安全性。一旦主密码泄露，攻击者将有可能计算出你的所有网站密码。

## 特色功能

### 提醒我更改主密码

为了最大程度地保障你的数字资产安全，Passtateless 内置了主密码定期更换提醒机制。  
你可以自行设置一个天数阈值（默认 180 天）。当距离你上次设置或更改主密码达到该天数时，Passtateless 会在你打开应用时主动要求你更改主密码。  
这能有效降低主密码长期不变带来的潜在安全风险，帮助你保持良好的安全习惯。

### 完全可配置的生成算法

不同于其他无状态密码管理器采用固定、黑盒的密码生成逻辑，Passtateless 将密码生成的控制权完全交给了你。  
Passtateless 支持使用 JSON 来精确定义密码生成的每一个计算步骤。你可以自由组合哈希函数、密钥派生函数、字符截取、大小写转换、插入特定符号等操作。  
我们也为你内置了一些开箱即用的预设，可以满足大部分常规要求。  

这意味着：
* **适应特殊要求**：无论目标网站对密码有什么奇葩的规则限制，你都能通过修改 JSON 配置适配。
* **绝对透明**：你确切地知道你的密码是如何一步步被计算出来的，没有隐藏的后门或不可预知的逻辑。

### 密码评估

在设置主密码或测试生成算法时，如何判断密码是否足够安全？Passtateless 为你提供了专业的答案。  
我们内置了业界知名的 `zxcvbn` 密码强度评估算法。  
与传统的“必须包含大小写字母和数字”这种表面化的规则不同，`zxcvbn` 通过模式匹配和概率计算，能够智能识别密码中的常见词汇、日期、键盘漫步模式等弱点，并给出真实破解时间估算。  
它可以帮你直观地评估你的主密码强度，确保你的安全基石坚不可摧。
